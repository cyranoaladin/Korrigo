# Coding Guidelines

## Python (Backend)
- **Style**: Follow PEP8 strictly.
- **Typing**: Use Type Hints for all function signatures.
- **Documentation**: All docstrings must be in **French** (The team is francophone).

## Vue.js (Frontend)
- **Component Style**: Use Composition API with `<script setup>`.
- **Structure**: Ensure components are atomic and reusable.
- **State Management**: Manage global state strictly with **Pinia**. Avoid complex prop drilling.

## Mathematics
- **Rendering**: All mathematical comments or text fields must support LaTeX rendering via **KaTeX**.
